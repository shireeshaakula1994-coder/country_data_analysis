# 🌍 Country Data Analysis Using REST API

## 📌 Project Overview
This project fetches live country data from the REST Countries API and performs basic data analysis using Python.

## 📊 Objectives
- Fetch data from public API
- Parse JSON response
- Create structured DataFrame
- Perform basic analysis on population and area

## 🛠️ Tools Used
- Python
- Requests library
- Pandas

## 📡 API Used
https://restcountries.com/v3.1/all

No authentication required.

## 📈 Analysis Performed
- Extracted country name, population, region, and area
- Found region with highest total population
- Identified country with largest area

## ▶️ How to Run
1. Open Jupyter Notebook or Google Colab
2. Run all cells in order
3. Ensure internet connection for API request

## 📌 Key Insights
- Population varies significantly across regions
- Russia/Canada-type countries dominate area size
